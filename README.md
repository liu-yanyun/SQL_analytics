# SQL_analytics

A collection of SQL analytics projects (monorepo) built as a reproducible portfolio.
Focus: relational design, joins, time-series aggregation, and analysis on real public datasets.

## Projects

| Project | Status | Skills showcased |
|---|---:|---|
| [01 Ireland Driving Test](projects/01_ireland_driving_test/README.md) | In progress | JOINs, public data, time-series aggregation |
| [02 Ireland Energy & Electricity](projects/02_ireland_energy/README.md) | Planned | Time-series, renewables contribution |
| [03 Life Science Expression](projects/03_life_science_expression/README.md) | Planned | Relational bio-data (samples/genes/expression) |
| [04 Movie Ratings](projects/04_movie_ratings/README.md) | Planned | Ranking, joins, window functions |

## How this repo is organized
- `projects/` — each project is self-contained with its own README + SQL/scripts
- `shared/` — templates/shared helpers (optional)
